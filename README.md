# 🌀 Vortex

## Organização Premium com Kanban

Vortex é um aplicativo web de gerenciamento de projetos estilo **Trello/Kanban**, desenvolvido com HTML, CSS, JavaScript e Firebase. Ele permite criar quadros, listas e cards, além de arrastar tarefas entre colunas e acompanhar o progresso com checklists.

---

## ✨ Funcionalidades

### 🔐 Autenticação
- ✅ Cadastro de usuários (email/senha)
- ✅ Login seguro
- ✅ Recuperação de senha por e-mail ("Esqueci minha senha")
- ✅ Sessão persistente

### 📊 Dashboard
- ✅ Métricas em tempo real (quadros, listas, cards, tarefas concluídas)
- ✅ Gráfico neural animado (conexões entre métricas)
- ✅ Cards com informações dos quadros

### 🎯 Quadros (Boards)
- ✅ Criar quadros personalizados
- ✅ Visualizar lista de quadros no dashboard
- ✅ Excluir quadros

### 📋 Listas (Lists)
- ✅ Criar listas dentro de cada quadro (ex: "A Fazer", "Em Andamento", "Concluído")
- ✅ Editar título da lista
- ✅ Excluir lista (remove todos os cards dentro)

### 🃏 Cards
- ✅ Criar cards dentro das listas
- ✅ Editar título e descrição
- ✅ Adicionar checklist (lista de tarefas)
- ✅ Marcar/desmarcar itens do checklist
- ✅ Excluir card

### 🔄 Drag & Drop
- ✅ Arrastar cards entre listas (mouse e toque)
- ✅ Mudança de status visual durante o arrasto

### ⚙️ Configurações
- ✅ Trocar senha
- ✅ Resetar todos os dados (apagar quadros)
- ✅ Excluir conta permanentemente

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Finalidade |
|------------|------------|
| **HTML5** | Estrutura da aplicação |
| **CSS3** | Estilização premium (cores laranja/rosa, fundo preto) |
| **JavaScript (ES6+)** | Lógica da aplicação, interações, drag & drop |
| **Firebase Authentication** | Login, cadastro, recuperação de senha |
| **Firebase Firestore** | Banco de dados em tempo real na nuvem |
| **Vercel** | Hospedagem da aplicação |
